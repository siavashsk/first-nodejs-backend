## Installation
1. Clone the repo
   ```sh
   git clone https://github.com/siavashsk/first-nodejs-backend.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the development server
    ```sh
   npm start
   ```
